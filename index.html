Metric Definitions & Logic

Avg. True Vulns: The number of unique, valid security flaws (e.g., SQLi, XSS) found in an application’s code or dependencies annually that require remediation.

Minutes to Triage: The time spent by a developer or AppSec engineer to open a ticket, investigate the code, and determine if a finding is a "True Positive" or a "False Positive."

App-Layer Incidents: Telemetry/alerts specifically targeting the application stack (OWASP Top 10) rather than network-level noise (e.g., port scans).

Mins to Verify/Triage: The duration from the moment a SOC alert fires to the point an analyst confirms it was a successful exploit attempt vs. a blocked/irrelevant probe.
